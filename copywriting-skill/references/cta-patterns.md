# CTA patterns — buttons + closes

The CTA is the smallest copy on the page and the most important. It's where intention becomes action. If the rest of the copy earned the read, the CTA's only job is to remove friction.

## Button copy — the 3 anatomy rules

1. **Verb-led.** Start with an action verb. "Book," "Read," "Reply," "Get," "Start."
2. **Specific.** "Book the 20-min call" beats "Contact us." "Read the 12-page brief" beats "Learn more."
3. **2-5 words.** Longer = friction. Shorter = vague.

## The 12 button formulas

Most strong button copy collapses to one of these:

| # | Pattern | Example |
|---|---------|---------|
| 1 | Verb + specific noun | Book the call |
| 2 | Verb + duration | Start the 14-day trial |
| 3 | Verb + named outcome | Get the audit |
| 4 | Verb + the noun (ownership) | Read the brief |
| 5 | Verb + count | Save my seat (12 left) |
| 6 | Verb + price (DTC) | Buy for $49 |
| 7 | Verb + format | Download the PDF |
| 8 | Verb + reader-state | Show me the data |
| 9 | First-person commitment | I want in |
| 10 | Verb + reversal | Try it free |
| 11 | Verb + low-friction qualifier | See pricing |
| 12 | Single-word punch | Apply |

## Buttons to avoid

- **"Click here"** — never. Tells the reader nothing.
- **"Learn more"** — vague. Replace with what they'll actually learn.
- **"Submit"** — clinical. Replace with what they're getting in return.
- **"Get started"** — empty. Replace with the specific first step.
- **"Sign up"** — friction-implying. Replace with the value they get.

## The CTA supporting line

Most buttons benefit from one line of supporting microcopy beneath them. The line removes the last objection.

| Pattern | Example |
|---------|---------|
| Risk reversal | No card needed. Cancel anytime. |
| Time commitment | 20 minutes. We do all the prep. |
| Scarcity (real, not fake) | 12 seats. Friday cohort. |
| Process clarity | We reply within 24 hours. |
| Social proof (specific) | 47 founders did this last quarter. |
| Outcome restatement | You leave with the deck. |

If the supporting line isn't load-bearing, cut it. White space is fine.

## The 3 closing copy moves

When the CTA needs more than a button — at the end of a sales letter, a landing page, a proposal — use one of these.

### Close 1 — The Sackheim guarantee restate
Bookend the letter with the exact same guarantee that opened it. Reader can re-read both bookends and feel the safety net.

> Try it for 30 days. If you don't see the change, we'll refund the entire fee. No questions, no friction. Just say the word.

### Close 2 — The Bernbach disqualifier
Disqualify the wrong reader. Earns trust and self-selects the right one.

> If you need a deck designer, we're not a fit — there are agencies that do that beautifully. We rewrite the narrative. If that's the gap, reply.

### Close 3 — The Trott binary
Two paths. One is named clearly. The reader picks.

> Reply if you want to see the teardown. Don't if your current deck is working. Either is fine.

## Urgency without hype

Real urgency works. Fake urgency reads as cheap.

| Real | Fake |
|------|------|
| "12 seats. Friday cohort fills." | "Limited time only!" |
| "Pricing changes Aug 1." | "Don't miss out!" |
| "We only take 3 clients a quarter." | "Act now!" |
| "The early-rate drops in 6 days." | "Hurry while supplies last!" |

The test: would you put a specific number, name, or date next to it? If yes, the urgency is real. If no, it's hype.

## CTA placement rules

- **Above the fold:** one CTA on the hero. Nothing else competing.
- **Mid-page:** one CTA after the proof block. Same wording as above-fold or a deepened version.
- **Bottom:** one CTA after the close. Final ask, often paired with the guarantee restate.
- **Never:** more than one CTA action per page. Multiple offers halve the conversion of both.

## The strip-button test

Cover the rest of the page. Show only the CTA button + supporting line.

Could a stranger guess what they'd be doing if they clicked? If yes, the button works. If no, rewrite.
