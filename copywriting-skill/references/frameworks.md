# Frameworks — index

A framework is the structural skeleton of a piece of copy. Pick ONE per draft. The framework doesn't dictate the words — it dictates the order the reader's mind walks through.

## Picker

| Copy task | Framework | Cluster file |
|-----------|-----------|--------------|
| Landing page, hero section | AIDA | `frameworks-attention.md` |
| Problem-led section, paid ad | PAS (Problem-Agitate-Solve) | `frameworks-attention.md` |
| Transformation pitch | BAB (Before-After-Bridge) | `frameworks-attention.md` |
| Section-level pattern, proposal section | 4Ps (Promise-Picture-Proof-Push) | `frameworks-sales.md` |
| Product detail, feature breakdown | FAB (Feature-Advantage-Benefit) | `frameworks-sales.md` |
| Long-form sales letter (cold list) | AIDCA | `frameworks-sales.md` |
| Long-form sales letter (warm/cold mix) | Schwartz 5-stage | `frameworks-awareness.md` |
| Long-form narrative, email body | Slippery Slide | `frameworks-awareness.md` |
| Executive summary, consultative pitch | SCQA (Situation-Complication-Question-Answer) | `frameworks-awareness.md` |

## How to pick

1. **Audience state.** Cold list → SCQA, Schwartz, or AIDCA. Warm → AIDA, BAB. Already-customer → just the offer.
2. **Length.** Short (under 100 words) → PAS or BAB. Medium → AIDA or 4Ps. Long → Schwartz 5-stage or AIDCA.
3. **Surface.** Hero section → AIDA. Section header → 4Ps. Product detail → FAB. Pitch deck → SCQA. Sales letter → AIDCA or Schwartz.
4. **Pain orientation.** If the reader already knows their pain, lead with the solve (BAB). If they don't, lead with the pain (PAS).

## When in doubt

- Hero/landing → **AIDA**.
- Section-level → **4Ps**.
- Long-form → **Schwartz 5-stage**.
- Pitch doc → **SCQA**.
- Feature breakdown → **FAB**.

## Framework declaration (mandatory before drafting)

Before writing, declare:

```
Framework: [name]
Audience state: [Unaware / Problem-Aware / Solution-Aware / Product-Aware / Most-Aware]
Length target: [words]
Surface: [headline / hero / section / email / sales letter / proposal]
```

If the framework doesn't fit the surface, pick a different framework, not a different surface.

## What frameworks are NOT

- **Not formulas to fill in.** A framework is a structural guide. The words are still craft.
- **Not interchangeable.** AIDA and BAB feel similar but route the reader differently. Pick the one that matches the audience state.
- **Not stackable in draft one.** Don't write "PAS for the section, then BAB in the close." Pick one. Layer later if needed.
