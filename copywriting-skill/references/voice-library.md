# Voice library — index

Voice is the irreducible signature of how a piece of copy sounds. Pick ONE per draft. No blending in draft one.

## Picker

| Situation | Voice | Cluster file |
|-----------|-------|--------------|
| Direct, verb-driven, energy | Trott / Bencivenga / Wieden | `voices-punch.md` |
| Minimalism, white space, restraint | Abbott / Krone | `voices-restraint.md` |
| Admit-the-flaw honesty, founder letters | Bernbach / Gossage | `voices-honesty.md` |
| DTC, sales letters, awareness-state work | Sugarman / Sackheim / Schwartz | `voices-dtc.md` |
| Challenger brand, zag-when-category-zigs | Hegarty / McElligott | `voices-challenger.md` |

## How to pick

1. **Read the brief.** What is the deliverable trying to do — punch, persuade, disarm, sell, surprise?
2. **Read the audience.** Skeptic? Tired? Curious? Loyal? Choose the voice that lands hardest on that audience.
3. **Read the brand register.** Luxury → restraint. Challenger → challenger. Direct response → DTC.
4. **Pick ONE.** Pull the cluster file. Don't blend until draft 2 or later.

## When in doubt — defaults

- Honesty needed → **Bernbach**.
- Energy needed → **Wieden**.
- Restraint needed → **Abbott**.
- Proof needed → **Bencivenga**.
- Speed/B2B → **Trott**.

When still in doubt: write it in Trott. Direct, verb-first, simple-interesting-relevant. You can layer voice later.

## Voice declaration (mandatory before drafting)

Before typing the first word of copy, declare:

```
Voice: [master name]
Register: [formal / casual / dry / warm / urgent / contemplative]
POV: [you-dominant / brand-we / third-person]
Cadence: [flat hammer / conversational / rhythmic punch / whisper / direct pitch]
```

If you can't fill those four lines, you're not ready to write.

## What voice is NOT

- Voice is not a "tone." Tone is a layer; voice is the structural fingerprint.
- Voice is not the brand's preference statement. Voice is the *technique* — sentence patterns, word choices, rhythm — that makes the copy sound like a specific master.
- Voice is not optional. Generic copy is voice-less copy. The whole point of this library is to refuse the default.
